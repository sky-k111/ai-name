# 需求规格说明书 — AI 智能取名系统

> 版本：3.0 | 日期：2026-07-04 | 作者：陈奕恺

---

## 1. 引言

### 1.1 项目背景

传统取名依赖长辈经验或翻查字典，效率低且受限于个人知识面。本项目利用 DeepSeek 大语言模型的语义理解与文化知识储备，为用户提供智能化的取名建议、名字分析、对比和精品取名服务。

### 1.2 项目范围

软件外包创新实践基地 2025 级新人实训项目，已完成难度 1 + 2 + 3 全部内容。

### 1.3 定义

| 术语 | 说明 |
|------|------|
| LLM | 大语言模型 |
| DeepSeek | 本项目使用的 AI 服务 |
| JWT | JSON Web Token，用户认证 |
| VIP/SVIP | 付费会员等级 |

---

## 2. 用户角色

| 角色 | 说明 | 权限 |
|------|------|------|
| 匿名用户 | 未登录 | 仅登录/注册页面 |
| 普通用户 | 已注册 | 取名、分析、对比、精品取名、收藏 |
| VIP | 15元/月 | 每日 10 次免费付费功能 |
| SVIP | 30元/月 | 每日 30 次免费付费功能 |
| 管理员 | 系统预设 | 全部功能 + 管理后台 |

---

## 3. 功能需求

### 3.1 取名模块

| 编号 | 功能 | 价格 | 说明 |
|------|------|------|------|
| F-01 | 基础取名 | 免费 | 表单 → AI 生成 5 个名字，支持多轮微调 |
| F-02 | 名字分析 | ¥1/次 | 输入名字 → AI 分析寓意/五行/评分 |
| F-03 | 名字对比 | ¥1/次 | 输入 2-5 个名字 → 多维度横向对比排名 |
| F-04 | 精品取名 | ¥2/次 | 八字分析 + 音律评分 + 字形评价 + 重名概率 |
| F-05 | 收藏夹 | 免费 | 收藏/取消收藏/查看收藏列表 |

### 3.2 付费体系

| 编号 | 功能 | 说明 |
|------|------|------|
| F-06 | 余额充值 | 10/20/50 元充值按钮（预留支付宝接口） |
| F-07 | VIP 会员 | 15元/月，每日 10 次免费 |
| F-08 | SVIP 会员 | 30元/月，每日 30 次免费 |
| F-09 | 每日体验 | 普通用户每天 1 次免费体验 |
| F-10 | 交易记录 | 充值/消费/VIP购买记录 |

扣费优先级：管理员免费 → VIP/SVIP免费用量 → 每日体验 → 扣余额

### 3.3 用户系统

| 编号 | 功能 | 说明 |
|------|------|------|
| F-11 | 注册 | 用户名+密码，密码强度指示，bcrypt 加密 |
| F-12 | 登录 | JWT 24h，支持用户名或邮箱登录 |
| F-13 | 绑定邮箱 | QQ邮箱验证码，60s 倒计时 |
| F-14 | 找回密码 | 邮箱验证码重置 |
| F-15 | 修改密码 | 需输入旧密码 |
| F-16 | 修改用户名 | 重名校验 |
| F-17 | 上传头像 | base64，≤1MB |
| F-18 | 注销账号 | 删除所有数据，认证日志保留 |
| F-19 | 个人统计 | 取名总次数 + 收藏数 |

### 3.4 历史记录

| 编号 | 功能 |
|------|------|
| F-20 | 四种类型：取名/分析/对比/精品 |
| F-21 | 类型筛选、展开详情 |
| F-22 | 单条删除、批量删除、一键清空 |

### 3.5 管理后台（仅管理员）

| 编号 | 功能 |
|------|------|
| F-23 | 用户列表（可查看详情、重置密码） |
| F-24 | 认证日志（注册/登录/登出/注销，含 IP 和时间） |
| F-25 | 全部取名记录（含已删除标记） |
| F-26 | 今日活跃用户 |

### 3.6 其他

| 编号 | 功能 |
|------|------|
| F-27 | 深色模式（太阳/月亮切换，刷新保持） |
| F-28 | 精品取名导出 PDF 报告 |
| F-29 | 语音试读名字 |
| F-30 | 密码显隐切换 |

## 4. 非功能需求

| 属性 | 要求 |
|------|------|
| 安全性 | bcrypt 密码、JWT 认证、角色权限、频率限制、登录保护 |
| 可用性 | Apple 极简风格，响应式适配 |
| 响应时间 | 生成 <15s，分析 <10s |
| 兼容性 | Chrome/Edge/Safari 最新版 |
| 数据库 | MySQL 5.7（Docker） |

---

## 5. 验收标准

1. 用户可注册、登录、登出
2. 四种取名模式均可用，结果正确展示
3. 付费体系完整：充值→VIP→扣费→余额不足提示
4. 管理后台可查看用户/日志/取名记录/重置密码
5. 未登录无法访问需登录页面
6. 非管理员无法访问管理后台
7. 深色模式切换正常且刷新保持
8. 所有错误状态有友好提示
