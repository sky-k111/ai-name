# 需求规格说明书 — AI 智能取名系统

> 版本：3.0 | 2026-07-13 | 陈奕恺

## 1. 项目概述

基于大语言模型的智能取名系统，支持基础取名、AI名字分析、多名字对比、精品取名。已完成实训难度1+2+3。

## 2. 用户角色

| 角色 | 权限 |
|------|------|
| 游客 | 可浏览首页，体验一次基础取名 |
| 普通用户 | 取名/分析/对比/精品取名/收藏/历史 |
| VIP(15元/月) | 每日10次免费付费功能 |
| SVIP(30元/月) | 每日30次免费付费功能 |
| 管理员(admin) | 全功能+管理后台 |

## 3. 功能需求

### 取名模块
| 功能 | 价格 | 说明 |
|------|------|------|
| 基础取名 | 免费 | 表单→AI生成5个名字+多轮微调 |
| 名字分析 | ¥1 | 输入名字→AI分析评分/寓意/五行/优缺点 |
| 名字对比 | ¥1 | 2-5个名字横向对比多维度排名 |
| 精品取名 | ¥2 | 八字分析+音律评分+字形评价+重名概率 |
| 收藏夹 | 免费 | 收藏/查看/取消收藏 |

### 付费体系
- 充值(10/20/50元) · VIP(15元/月) · SVIP(30元/月)
- 每日1次免费体验 · 管理员全部免费
- 扣费优先级：管理员 → VIP/SVIP免费用量 → 体验 → 余额

### 用户系统
- JWT注册登录 · 邮箱或用户名登录 · QQ邮箱验证码绑定
- 找回密码 · 修改密码/用户名 · 上传头像 · 注销账号
- 密码强度指示 · 个人取名统计

### 历史记录
- 4种类型筛选 · 展开详情 · 分析记录显示分数
- 单条/批量/一键清空

### 管理后台
- 总用户数/取名次数/今日活跃(可点击查看详情)
- 认证日志 · 全部取名记录 · 重置用户密码

### UI
- 宣纸风格 · 深色模式 · 抽屉式工作台 · 移动端适配

### 安全
- bcrypt密码 · 频率限制 · 登录5次封15分钟

## 4. 非功能需求
- 数据库：MySQL(Docker) · 响应时间<15s · Chrome/Edge兼容

## 5. 验收标准
1. 四种取名模式正常 2. 付费体系完整 3. 管理后台可用
4. 未登录拦截 5. 权限分级正确 6. 所有错误有提示
