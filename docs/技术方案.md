| 邮件 | QQ邮箱 SMTP | - | smtplib |

## 3. 核心模块

### 3.1 后端

| 模块 | 文件 | 职责 |
|------|------|------|
| 取名服务 | naming_service.py | Prompt 拼装、DeepSeek 调用、JSON 解析 |
| 付费服务 | payment_service.py | 充值/扣费/VIP/次数检查/日使用量 |
| 认证服务 | auth_service.py | 密码哈希、JWT 签发、管理员初始化 |
| 邮件服务 | email_service.py | QQ邮箱 SMTP、验证码生成/验证/频率限制 |
| 认证中间件 | middleware/auth.py | JWT 解析、用户注入、管理员检查 |
| 安全中间件 | middleware/security.py | IP 频率限制、登录暴力防护 |

### 3.2 前端

| 页面 | 功能 |
|------|------|
| HomeView | 四种模式切换（取名/分析/对比/精品）+ 收藏 + 微调 |
| LoginView | 登录/注册/找回密码/密码强度 |
| ProfileView | 个人中心/修改密码/绑定邮箱/头像/统计/注销 |
| HistoryView | 历史列表/类型筛选/展开详情/批量删除/一键清空 |
| BalanceView | 钱包充值/买VIP/交易记录 |
| FavoritesView | 收藏夹 |
| AdminView | 管理后台：用户列表/日志/取名记录/今日活跃 |

## 4. 数据库（MySQL 7 张表）

- **users**：用户 + 余额 + VIP + 日使用量 + 头像
- **naming_history**：取名/分析/对比/精品记录 + 软删除 + 类型标记
- **auth_logs**：注册/登录/登出/注销日志
- **email_verifications**：邮箱验证码（一次性，10分钟过期）
- **transactions**：充值/消费/VIP购买记录
- **daily_usage**：VIP 每日免费次数记录
- **favorites**：收藏夹

## 5. 安全措施

| 措施 | 实现 |
|------|------|
| 密码加密 | bcrypt 单向哈希 |
| JWT 认证 | 24h 过期，sub 字符串 |
| 角色权限 | user/admin，路由守卫 + 中间件 |
| 频率限制 | IP 每分钟 60 次（内存） |
| 登录保护 | 连续失败 5 次封 IP 15 分钟 |
| 密钥检查 | 启动时检测默认 SECRET_KEY |
| SQL 注入 | SQLAlchemy 参数化查询 |
| CORS | 仅允许前端域名 |

---

## 6. Prompt 工程

- System Prompt：取名大师角色 + JSON 格式约束 + 禁止值内出现英文双引号
- 精品取名额外字段：bazi、sound_analysis、char_analysis、popularity
- 对比命名：多维度评分（寓意/音韵/五行/字形）+ 排名
- JSON 解析三级降级 + 中文引号替换容错
