# 编译构建安装部署手册 — AI 智能取名系统

> 版本：3.0 | 日期：2026-07-04

---

## 1. 环境要求

| 软件 | 版本 | 说明 |
|------|------|------|
| Python | 3.10+ | 后端 |
| Node.js | 18+ | 前端构建 |
| Docker | 20+ | 运行 MySQL |
| DeepSeek API Key | — | platform.deepseek.com |

---

## 2. 获取源码

```bash
git clone <仓库地址>
cd ai-naming
```

或解压 `src.zip` / `deploy.zip`。

---

## 3. 数据库（Docker MySQL）

```bash
docker run -d --name ai-naming-mysql \
  -e MYSQL_ROOT_PASSWORD=rootpassword \
  -e MYSQL_DATABASE=naming_db \
  -p 3306:3306 \
  mysql:8.0 --character-set-server=utf8mb4 --collation-server=utf8mb4_unicode_ci
```

---

## 4. 后端部署

### 4.1 安装依赖

```bash
cd backend
pip install -r requirements.txt
```

依赖包括：fastapi, uvicorn, httpx, pydantic, python-dotenv, sqlalchemy, pymysql, python-jose[cryptography], passlib[bcrypt], bcrypt

### 4.2 配置

复制并编辑 `.env`：

```ini
DEEPSEEK_API_KEY=sk-your-key
DATABASE_URL=mysql+pymysql://root:rootpassword@localhost:3306/naming_db
SECRET_KEY=your-production-secret-key
ADMIN_USERNAME=admin
ADMIN_PASSWORD=your-password
SMTP_HOST=smtp.qq.com
SMTP_PORT=587
SMTP_USER=your@qq.com
SMTP_PASSWORD=授权码
SMTP_FROM=your@qq.com
```

### 4.3 启动

```bash
uvicorn app.main:app --host 127.0.0.1 --port 8000 --workers 4
```

验证：`curl http://localhost:8000/api/health` → `{"status":"ok"}`

## 5. 前端部署

### 5.1 安装依赖

## 6. 一体化部署

### Nginx 配置

```nginx
server {
    listen 80;
    server_name _;

    location / {
        root /path/to/frontend/dist;
        try_files $uri /index.html;
    }

    location /api {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### Dockerfile

```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY backend/requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY backend/ .
COPY frontend/dist/ ./static/
EXPOSE 8000
CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]
```

## 7. 故障排查

| 问题 | 解决 |
|------|------|
| MySQL 连接失败 | 检查 Docker 是否运行：`docker ps` |
| 表不存在 | 表在应用启动时自动创建 |
| DEEPSEEK_API_KEY 未设置 | 创建 `.env` 或设置环境变量 |
| 前端 404 | 后端未启动 |
| CORS 错误 | Vite proxy 未生效 |
| AI 504 | DeepSeek 超时，重试 |
| pip install 失败 | Python 版本 < 3.14 |
| 登录 401 | token 过期（24h），重新登录 |
| Docker daemon 不运行 | 启动 Docker Desktop |
